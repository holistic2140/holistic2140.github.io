A Holistic Approach to Improve Learning and Motivation in CS1

Introduction

This project, based at Georgia Gwinnett College, integrates high-impact practices and automated tools to create a holistic approach for improving student learning and motivation in the introductory programming course (CS1). The primary goal is to address the challenges of high failure rates and low student retention, particularly among underrepresented groups in the IT field.

The Challenge

The introductory programming course, ITEC-2140 (Programming Fundamentals), has been identified as a significant bottleneck for students, with high failure rates and low continuation to subsequent courses. This project aims to reverse this trend by creating a more engaging and supportive learning environment.

Our Holistic Approach

We employ a multi-pronged strategy that caters to diverse learning styles. Our approach is structured into three parallel threads:

  • The Main Thread: Core learning activities including Process-oriented Group Inquiry Learning (POGIL), auto-graded tracing exercises (TracingQuiz), and the Iterative Problem-Solving Strategy Detection and Development (IPSSDD).
  • The Peer-driven Visual Programming Thread: A library of peer-recorded videos introducing concepts in a visual programming environment.
  • The Game Development Thread: A semester-long, team-based game development project for hands-on learning.
